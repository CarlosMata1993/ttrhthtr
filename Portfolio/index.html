Sobre mí

Hola, mi nombre es Carlos Ernesto Mata García, soy un entusiasta del diseño web, el dibujo, la música, la escritura... el arte en general.

Actualmente estoy aprendienddo programación con javascript y python, siempre un poco enfocado en el aspecto de expresión creativa que ésta permite. Aunque todavía estoy empezando, en el futuro espero poder aplicarlos de manera que pueda integrar y combinar mis intereses y crear historias y experiencias como las que a mi me inspiraron a intentar aprender todo esto.

Movie posters

Este fue uno de los primeros proyectos que realizé. Su objetivo era el de aprender a usar el Grid en css para crear una página visualmente llamativa y asimétrica. Añadí el efecto visual de la opacidad para darle un toque más interactivo en su momento, aunque todavía tengo planes de añadir más interactividad con javascript, como por ejemplo, reproducir un pequeño video de la pelicula correspondiente al hacer click sobre el poster.

Card slider

Este proyecto lo realizé siguiendo un tutorial, ya que el efecto es bastante impresionante visualmente. Se trata de crear un efecto 3D con "cartas" que son contenedores div moviendose en el eje z. Para ello, se utiliza una variable que es el número de contenedores, la cual luego se utiliza en una fórmula para calcular la posición de cada contenedor en el eje z.

Por experimentar un poco, aparte de lo explicado en el tutorial, he añadido que las cartas se detengan al pasar el ratón sobre ellas, y he cambiado la animación para que cuando se vea en dispositivos mas pequeños se ajuste al tamaño adecuado.

Info cards

Con este proyecto mi objetivo era el de seguir aprendiendo efectos 3D con css. Fue un buen ejercicio para practicar poner las posiciones correctamente a cada elemento, y hacer que las imagenes se vieran correctamente sobre las cartas. Además, le añadí un poco de interactividad con javascript para voltearlas.

Flipbook

Con este proyecto mi objetivo fue aplicar lo estudiado anteriormente con los efectos 3D y el posicionamiento correcto de los elementos, utilizando luego javascript para poder controlar las posiciones de las páginas dinámicamente. Aquí también llegue a un punto en el que tuve que seguir un tutorial para poder entender bien la lógica de javascript que era necesaria para hacer que el libro funcionara correctamente.
